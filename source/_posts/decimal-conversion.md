*   **其它进制转十进制**

例：一个x进制的数：abcde

转换过程：  a                   b                 c                 d                 e
```
a\*x^4    +    b\*x^3   +    c\*x^2   +   d\*x^1   +   e*x^0
```
(x^a意思为x的a次方)

例：16进制的 1FA2
```
1FA2 = 2\*16^0 +  A\*16^1 + F\*16^2 + 1\*16^3 = 2\*16^0 +  10\*16^1 + 15\*16^2 + 1\*16^3 = 8098
```
*   **十进制转其它进制**

这里利用短除法

例如115转换为8进制

转换过程：

115   /   8  =  14 ............ 3

14   /   8  =  1 ............. 6

1   /   8  =  0.............. 1

然后把余数倒过来写，即从下往上写，得到8进制的115为163，其它进制类似，只是除数不同而已。
